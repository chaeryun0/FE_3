<summary> slice(), splice(), sort() </summary>
<div markdown="1">
  
  ```
 let arr = [1, 2, 3, 4, 5]
 arr.splice(0, 1, 2)       // [1]
 arr.slice(3, 4)           // [4]
 arr.sort((a, b)=>a-b)     // [1, 2, 3, 4, 5]
 arr.sort((a, b)=>b-a)     // [5, 4, 3, 2, 1]
  ```
 ``` 
splice(0, 1, 2)는 (시작 index, 삭제 count, 추가 데이터)
  
→ 0번 인덱스부터 1개를 삭제하여 1만 삭제되고 그 자리에 추가 데이터인 2가 들어오게 된다. 
→ 반환 값은 삭제된 1, 원본 배열이 [2, 2, 3, 4, 5]로 변경됨

slice(3, 4)는 (시작 index, 마지막 index)
  
→ 3번 인덱스부터 4번 인덱스 "앞" 까지 배열로 반환
→ 반환 값 4, 원본 배열은 변경되지 않음

sort((a,b) => a-b) // 오름차순
sort((a,b) => b-a) // 내림차순
 ```
